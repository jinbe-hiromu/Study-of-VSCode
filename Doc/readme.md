# VSCode
## ショートカット

| key         | mean                          |
| ----------- | ----------------------------- |
| Shift+Alt+F | コードの整形                        |
| Ctrl+Enter  | 行追加（Enterとの違いはカーソルが途中にあっても可能） |
| Ctrl+1      | ターミナル→エディタへの遷移                |
| Ctrl+Shift+@ |  エディタ→ターミナルへの遷移(macOSのみ)   |
